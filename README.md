# 薄膜沉积模拟
1. 溅射原子的产生：  Ar离子轰击靶材，产生沿不同方向不同速度分布的溅射原子
2. 溅射原子的传输：  溅射原子在传输过程中与Ar原子碰撞，速度的角向分布改变
3. 溅射原子的沉积： 溅射原子达到薄膜表面并被吸附，其中涉及到了原子吸附，原子扩散，再发射（第一次没被吸附，后面接着撞击）
